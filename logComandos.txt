COMANDOS BASH ou TERMINAL VSCODE

Configuração global para assinatura nos projeto
- git config --global user.name "John Kleber"
- git config --global user.mail "jo.kleber85@gmail.com"

Comandos / ações 
- mkdir projeto (criando pasta)
- touch "nome arquivo" (criando arquivo)

- git init (inicialização repositório)
- git add . (add todos os arquivos para serem monitorados pelo git)
- git add "nome do arquivo" (add arquivo específico para ser monitorado pelo git)
- git commit -m "descriçao do arquivo ou alterações" 
- git log (vizualizar o histórico do projeto)


- git clear (limpar tela)
